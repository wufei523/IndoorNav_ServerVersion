IndoorNav Server Version
==========

This repository contains the server version for IndoorNav. IndoorNav is a indoor
 navigation system based on D* lite.

Copyright (C) 2016 Fei Wu

Content of this Distribution:
-----------------------------

bin/           Java class files
myMap/         Map files
src/           Java source files
build.xml      Ant build file for IndoorNav
IndoorNav.sh   Command line script to start the program
README         This file
wholeMap.png   Map in png format


Build:
----------------

Use ant build file to build


Getting Started::
----------------

If you just like to run IndoorNav in command line mode, try the following:

Go to the folder 'IndoorNav_ServerVersion'.
Execute IndoorNav.sh (Unix, Linux, and Mac OS).
